# Base de Conhecimento de Operações & Logs de Erro

Esta pasta é mantida de acordo com o `Protocolo de Erro 360°`.

**Formato Obrigatório para Adicionar Erros Resolvidos:**
- **Erro:** [Descrição literal do erro]
- **Contexto:** [O que estava sendo feito]
- **Causa Raiz:** [O motivo causador do erro]
- **Solução:** [Correção ou comando executado]
- **Prevenção:** [Como garantir que não ocorra novamente]
