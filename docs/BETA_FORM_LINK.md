# Links do Google Forms (Bug Report)

Como sou uma IA, não consigo criar o Google Form na sua conta autenticada automaticamente. 
**Ação Necessária (2 minutos):**
1. Acesse [Google Forms](https://forms.google.com) e crie um novo formulário chamado "Relatar Bug — Rootline Beta".
2. Adicione as seguintes perguntas:
   - **Nome:** (Resposta curta, Obrigatório)
   - **WhatsApp ou Email:** (Resposta curta, Obrigatório)
   - **Tipo de problema:** (Múltipla escolha ou Lista suspensa)
     - Login / Cadastro
     - Criar família / Onboarding
     - Timeline
     - Tree / Árvore
     - Convite / Aceite de convite
     - Perfil
     - Tema (claro/escuro) / Legibilidade
     - Outros
   - **O que você fez?** (Parágrafo)
   - **O que você esperava?** (Parágrafo)
   - **O que aconteceu?** (Parágrafo)
   - **Envio de imagem (print):** (Upload de arquivo)
   - **Qual dispositivo?** (Lista suspensa: Android / iPhone / PC)
   - *A data e hora são capturadas automaticamente pelo Google Forms.*
3. Gere o link de compartilhamento e crie a planilha de respostas.
