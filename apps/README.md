# Aplicativos (Apps)

Neste diretório ficam os módulos principais de nível de aplicação do projeto RootLine. Diferentes serviços, como aplicações frontend SPA/SSR e backends, devem ser criados em diretórios individuais aqui seguindo a estrutura monorepo.

*Nota: Garanta que cada serviço inicializado em `apps/` siga as decisões de arquitetura em `docs/adr/`.*
